# Event-Based-Traffic-Simulation
Urban traffic becomes highly unpredictable during large events like concerts and football games. Traditional routing systems don’t capture dynamic congestion caused by these events.
This project is a web-based **Traffic Digital Twin** that integrates:

- Real-time traffic routing (TomTom API)
- Event-based congestion simulation (concerts, football games)
- Animated vehicle flow using Leaflet
- Multiple route visualization

---

## 🌍 Features

- 🚦 Real-time traffic-aware routing
- 🛣️ Multiple route alternatives
- 🚗 Animated vehicles with direction
- 🎯 Event-based congestion (stadium simulation)
- 📊 Dynamic travel time estimation

---

## 🧠 Tech Stack

- Frontend: HTML, JavaScript, Leaflet
- Backend: Python (Flask)
- APIs:
  - TomTom Traffic API
  - OpenStreetMap
